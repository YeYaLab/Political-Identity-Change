#### **Dataset Description**
This dataset contains matrices of neural activity for each participant and each video clip used in the study.  
Only voxels located within **grey matter** are included.

Each matrix corresponds to:

- A **single participant**
- A **specific video clip**
- Time series or summary data for **all grey-matter voxels**
