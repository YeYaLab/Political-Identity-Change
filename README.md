# Data and Code Availability

This repository contains the data and code associated with the following paper:

**Boiman, G., Ohad, T., Zvi, Y., Katabi, N., & Yeshurun, Y. (in press).**  
*Neural correlates of political attitudes identity change: A longitudinal fMRI study.*  
**Communications Psychology.**

The repository includes:

- Scripts for analysis  
- Links to NIfTI maps and neural data matrices
